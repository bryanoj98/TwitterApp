# APP para la gestion de Twitter
Rapid es una aplicacion intermediaria entre el usuario y la red social Twitter, desarrollada en Nodejs. Capaz de listar Seguidos, Segidores y personas que no te siguen de vuelta. Ademas, cuenta con la posibilidad de dejar de seguir.

## Pre-requisitos

Se deben instalar las librerias y demas dependencias necesarias para la 
correcta ejecucion del proyecto
```sh
npm install
```
